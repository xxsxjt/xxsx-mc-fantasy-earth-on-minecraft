Development test build for Minecraft 26.2 / NeoForge 26.2.0.7-beta.

- Registers `Attributes.FLYING_SPEED` because `ManaWispEntity` uses `FlyingMoveControl` and `FlyingPathNavigation`.

- Preserves the alpha.6 icon and all alpha.5 gameplay, persistence and standalone journey behavior.
