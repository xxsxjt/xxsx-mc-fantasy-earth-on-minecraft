# Fantasy Earth on Minecraft 0.7.0-alpha.1

> Development preview for Minecraft 26.2 and NeoForge 26.2.0.7-beta. Content, balance and saves may change before a stable release.

## Highlights

- Renamed the public mod identity to `Fantasy Earth on Minecraft / 魔幻地球 on Minecraft`.
- The mod is independently playable; Earth on Minecraft, Earth Human and Xuanhuan Earth on Minecraft are optional integrations only.
- Added three contractable familiars, the crystal-armored spider, three resident roles and three generated arcane settlements.
- Added arcane panel, focus-mat attunement, chunk aether fields, distinct stations, ritual multiblock checks, JEI categories and bilingual handbook guidance.
- Fixed tagged machine inputs so vanilla fallback materials remain accepted and visible in JEI.
- Added safe settlement spawn-point selection and corrected witch-hamlet resident positions.

## Verification

- Standalone dedicated server reached `Done` with no optional ecosystem mods installed.
- Combined dedicated server reached `Done` with Xuanhuan Earth on Minecraft.
- All 12 ecosystem entities summoned successfully in the combined test.
- Three xuanhuan and three fantasy settlement features placed successfully; each route produced the expected nine residents.
- Combined client completed OpenGL initialization and resource reload without missing model or texture errors.
