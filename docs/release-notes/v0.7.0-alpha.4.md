# Fantasy Earth on Minecraft 0.7.0-alpha.4

Development test build for Minecraft 26.2 / NeoForge 26.2.0.7-beta.

- Makes endurance progression reduce actual food and saturation loss.
- Makes body-ward progression reduce entity-sourced combat damage and fall impact.
- Makes breathing research reduce underwater air consumption with a persistent multiplier.
- Defers shared body-adaptation consumption to Xuanhuan Earth when both mods are installed, preventing duplicate effects.
- Replaces write-only action NBT with server-broadcast player action payloads for attunement, mana recovery and active spells.
- Adds live unlock, level, XP and next-step guidance to the handbook start page.
- Synchronizes focus-mat HUD stages with the authoritative server cooldown instead of entity age.
- Keeps all integrations optional and preserves standalone vanilla-material routes.

Artifact: `fantasy-earth-on-minecraft-neoforge-26.2-0.7.0-alpha.4.jar`

Verification limit: source, resources and offline build are automated; client layout, player poses and gameplay effects still require an explicitly authorized in-game smoke test.
