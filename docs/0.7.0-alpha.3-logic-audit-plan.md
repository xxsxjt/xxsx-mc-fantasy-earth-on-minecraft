# 魔幻地球 on Minecraft 0.7.0-alpha.3 逻辑审查与修复计划

## 1. 本轮目标

本轮优先修复已有魔法路线的职责混乱、无成长和无动作问题，不继续用更多注册物掩盖未闭环功能。写入范围仅限 `earth_online_magic`；其他项目只读。

运行验证仅做源码、资源、离线构建、JAR 与日志检查。除非用户明确说“单独运行”，不启动或控制 Minecraft、启动器、Blender 或其他桌面程序。

## 2. 已确认问题

1. 魔幻田野手册职责基本正确，但合成奥术启蒙会消耗手册，导致新手导航入口丢失。
2. 奥术启蒙札记同时负责学习、调律、回法和身体恢复；护身/水肺札记同时是研究物和免费主动法术。
3. `ArcanePractice` 会在未研究时自动解锁奥术启蒙，研究物不是实际前置。
4. 研究与两条适应路线永远停在等级 1，重复调律没有经验、升级或长期收益。
5. 札记没有基础研究前置校验，玩家可以绕过启蒙直接掌握高级回路。
6. `last_action` 只写持久数据，没有形成动作表现。
7. 使魔只有通用走路/攻击 AI；坐姿、攻击时序和不同生物的动作没有完整渲染。
8. 魔力微灵沿用地面近战动物逻辑，视觉上悬浮但行为语义仍像普通四足动物。
9. 手册、tooltip、README 与 metadata 仍把研究物描述为重复回法/施法入口。

## 3. 修复契约

### 3.1 物品职责

- 魔幻田野手册永久保留，只负责导航和状态说明，不作为一次性研究耗材。
- 奥术启蒙、护身与水肺札记只负责首次研究；已研究后选择对应回路并打开奥术面板。
- 奥术面板/专注垫是重复调律和研究成长入口。
- 主动回路通过单独按钮与可配置快捷键释放，服务端校验法力、研究前置和冷却。

### 3.2 成长闭环

以太调律、护身和呼吸回路分别保存等级与经验。有效调律按以太场和辅助效率推进研究；升级增加对应的魔力上限、调律率、耐力、防护或呼吸收益。连续专注垫结算保持安静，只在手动操作、失败原因或升级时突出提示。

### 3.3 动作与生物

- 主动法术广播玩家动作、几何粒子轨迹和音效，实际命中与消耗由服务端处理。
- 使魔同步攻击进度与坐姿；以太狐、符文狼和魔力微灵采用不同节奏。
- 符文守望者和晶甲蜘蛛使用真实攻击进度驱动重击/扑击，而不是持续保持一个笼统战斗姿势。
- 魔力微灵后续迁移为真正悬浮导航；本轮至少修正重力、落地伤害和悬浮动作语义。

## 4. 静态验收

1. 手册不会因制作研究札记而被消耗。
2. 未研究启蒙不能通过面板、专注垫或高级札记绕过前置。
3. 有效调律推进当前回路经验，升级产生持久且可说明的收益。
4. 面板显示当前回路等级、经验和主动法术冷却。
5. 实体动作由攻击进度/坐姿等真实状态驱动。
6. 中英语言、手册、JEI、配方、README 与元数据一致。
7. 离线构建、JAR 内容和有效版本部署检查通过。
