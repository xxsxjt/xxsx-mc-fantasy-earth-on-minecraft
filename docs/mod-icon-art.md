# Mod icon art

- Provider/model: secured Image-2 workflow using `gpt-image-2`.
- Final resource: `neoforge-26.2/src/main/resources/logo.png`, 512 x 512 PNG.
- Visual identity: a block-shaped Earth inside a brass arcane astrolabe, with a violet aether crystal and restrained cyan circuit light.
- Reference assets: the existing field arcane notebook and alchemy table front were used only for palette and material language.
- Constraints: no text, no watermark, no generic MMO crest, and a silhouette distinct from the xuanhuan cultivation ring at 64 x 64.
- The original generated master remains in the local generated-image archive; the repository contains the reviewed downscaled release asset.
